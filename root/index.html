NestJS Concepts Overview

1. Decorators

Metadata annotations for classes and methods.

@Controller()
@Get(), @Post()
2. Middleware

Run before route handlers (like Express middleware).

app.use(loggerMiddleware);
3. Interceptors

Logic before/after request processing.

@UseInterceptors(LoggingInterceptor)
4. Pipes

Transform and validate input data.

@Body(new ValidationPipe()) dto: CreateUserDto
5. Guards

Control request access (e.g., authentication).

@UseGuards(AuthGuard)
6. Filters

Catch and handle exceptions.

@UseFilters(HttpExceptionFilter)
7. Lifecycle Hooks

Methods called at specific times during the app's lifetime.

onModuleInit() { ... }
